IO ModuleÔÉÅ

This module contains the functions pew uses for importing and exporting data.

AgilentÔÉÅ

Import of line-by-line collected Agilent ‚Äò.b‚Äô batches. Both raw binaries and the ‚Äò.csv‚Äô exports are supported. Tested with Agilent 7500, 7700 and 8900 ICPs.

Finds ‚Äò.d‚Äô datafiles in a directory.

pewlib.io.agilent.load_info(path)ÔÉÅ

Reads information from a batch.

Instrument info is read from the first Devices.xml found, batch info from the BatchLog.xml. An empty dictionary is returned if neither file can be read.

Possible keys:
